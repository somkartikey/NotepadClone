Notepad Clone
=============
A simple text editor built using Java, designed to mimic the functionality of Windows Notepad.

Features:
---------
- Create, open, and save text files
- Basic text editing (cut, copy, paste, undo, redo)
- Simple and user-friendly UI
- File handling for efficient text management

Requirements:
-------------
- Java Development Kit (JDK) 8 or later
- Any Java IDE (e.g., IntelliJ IDEA, Eclipse, or NetBeans)

How to Run:
-----------
Using Command Line:
1. Clone the repository:
   git clone https://github.com/somkartikey/NotepadClone.git

2. Navigate to the project directory:
   cd NotepadClone

3. Compile the Java files:
   javac NotepadClone.java

4. Run the application:
   java NotepadClone

Using an IDE:
1. Open the project in your preferred IDE.
2. Ensure JDK is properly set up.
3. Run the `NotepadClone.java` file.
